# MGS.LogHub

## Summary

- Hub for Unity project output log.

## Ability

- Output log to persistent data path.
- Set output log level.
- Auto clear log file period days.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.LogHub.git?path=/Assets
  ```
## Usage

- Unity menu "Tools/Log Hub/Settings" to set parameters.
- Unity menu "Tools/Log Hub/Outputs" to open output directory.
- Use UnityEngine.Debug to output log.
